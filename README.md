# AI File Renamer

Intelligent file renaming powered by Claude AI

## Features

- Drag & drop file upload
- AI-suggested filenames
- Owner/person name labeling
- Pattern-based renaming
- Batch processing
- Instant download

## Setup

1. Clone repository
2. Run `npm install`
3. Copy `.env.example` to `.env`
4. Add your Claude API key
5. Run `npm run dev`

## Deployment

- Deploy to Vercel with one click
- Set environment variable in Vercel dashboard
- Domain: file-renamer.vercel.app (or custom)

## Usage

1. Drag files into the drop zone
2. Click "AI Suggest Names" for intelligent suggestions
3. Edit names and add owner labels as needed
4. Download all files with new names
